# Network Reconnaissance Tool

A powerful, multi-threaded port scanner with banner grabbing for Kali Linux.

## Features

- Fast Multi-threaded Scanning
- Banner Grabbing
- Ping Sweep
- Colorized Output
- Custom Port Ranges
- Service Detection

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/network-recon-tool.git
cd network-recon-tool
chmod +x setup.sh
./setup.sh
